RESTFull API Web Web Service

Description URL
Use GET Method
Display All Infomation - http://localhost/restfull/horizonstudents/read.php
Specific Infomation - http://localhost/restfull/horizonstudents/read.php?idnum={enter ID Number}
Example:-
http://localhost/restfull/horizonstudents/read.php?idnum=5


Use Post man Tool For Following

Description URL
Use POST Method
Insert Data - http://localhost/restfull/horizonstudents/create.php
Insert the values according to the API
Use PUT Method
Update Data - http://localhost/restfull/horizonstudents/update.php?idnum={enter ID Number}
Insert the values To be UPDATED according to the API
Example:-
http://localhost/restfull/horizonstudents/read.php?idnum=5
Use DELETE Method
Delete Data - http://localhost/restfull/horizonstudents/delete.php?idnum={enter ID Number to be DELETED}

Example:-
http://localhost/restfull/horizonstudents/read.php?idnum=5
